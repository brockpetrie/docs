## Functions
### getTokenIdByTarget

Returns the token ID (if it exists) for an NFT that may have been patched

*Requires reverse storage enabled*


```solidity
function getTokenIdByTarget(address target) external returns (uint256 tokenId);
```
**Parameters**

|Name|Type|Description|
|----|----|-----------|
|`target`|`address`|Address of the original account|

**Returns**

|Name|Type|Description|
|----|----|-----------|
|`tokenId`|`uint256`|ID of the newly minted token|
