## Functions
### getTokenIdByTarget

Returns the token ID (if it exists) for an NFT that may have been patched

*Requires reverse storage enabled*


```solidity
function getTokenIdByTarget(PatchTarget memory target) external view returns (uint256 tokenId);
```
**Parameters**

|Name|Type|Description|
|----|----|-----------|
|`target`|`PatchTarget`|Patch to target of patch|

**Returns**

|Name|Type|Description|
|----|----|-----------|
|`tokenId`|`uint256`|token ID of the patch|
